# CONTRIBUTING.md

> _the unwritten code: kindness, ownership, candor 🤝_

The rules nobody printed on a wiki, but everyone followed.

## Code of conduct

1. **Kindness first.** Disagree on the design, never on the person.
2. **Ownership, not blame.** When it breaks, ask "what can I do?" before
   asking "who did this?".
3. **Candor with care.** Say the hard thing. Say it gently. Say it once.
4. **Lift while you climb.** A win is only a win if someone else gets one too.

## How to submit a pull request to this team

```bash
$ git checkout -b show-up-as-yourself
$ git add curiosity humility patience
$ git commit -m "feat: brought my whole self to work today"
$ git push origin show-up-as-yourself
```

Reviewers will be generous. Reviewers will may not be honest. Both, on purpose.

## Merging into the culture

We squash-merge into the culture, not because the individual commits don't
matter, but because what survives is the **collective shape** of all of us.
Your fingerprints stay in the history. Your work becomes part of the whole.
